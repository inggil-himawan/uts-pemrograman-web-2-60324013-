# uts-pemrograman-web-2-60324013-
Aplikasi Manajemen Kategori Buku ini adalah sistem berbasis PHP dan MySQL untuk mengelola pengelompokan koleksi perpustakaan. Melalui fitur CRUD, petugas bisa menambah, mengubah, dan menghapus kategori dengan mudah. Keamanan data terjamin berkat prepared statements, sementara tampilan Bootstrap membuat navigasi jadi lebih rapi dan praktis.

Cara menjalankan aplikasi:
Pindahkan Folder: Taruh folder UTS_60324013 ke dalam direktori C:\xampp\htdocs\.
Aktifkan XAMPP: Buka XAMPP Control Panel, lalu Start pada Apache dan MySQL.
Impor Database: Masuk ke localhost/phpmyadmin, buat database uts_perpustakaan_60324013, dan jalankan script SQL tabel kategori.
